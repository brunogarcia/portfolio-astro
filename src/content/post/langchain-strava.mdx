The [notebook](https://github.com/brunogarcia/langchain-strava-api/blob/main/strava_api.ipynb) is able to respond to questions about an athlete's activity.

Example:

> How many kilometres had run the user in his last activity?

## Requirements

- You must have a OpenAPI key
  - https://platform.openai.com/account/api-keys
- You must have an account on Strava
  - https://www.strava.com
- You must create an app on Strava
  - https://www.strava.com/settings/api
